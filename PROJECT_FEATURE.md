# AI Token Shop 项目功能列表

## 用户端功能模块
| 模块 | 核心功能 | 优先级 |
|------|---------|--------|
| 用户系统 | 注册/登录、个人资料、安全设置、消息通知 | ⭐⭐⭐⭐⭐ |
| 商品展示 | 商品分类、搜索筛选、详情页、价格对比 | ⭐⭐⭐⭐⭐ |
| 购物车 | 添加商品、数量调整、优惠券使用、结算 | ⭐⭐⭐⭐⭐ |
| 订单管理 | 订单列表、订单详情、退款申请、评价 | ⭐⭐⭐⭐ |
| 卡密管理 | 卡密购买、卡密查询、使用记录、批量操作 | ⭐⭐⭐⭐⭐ |
| 支付系统 | 多支付方式、订单支付、支付记录 | ⭐⭐⭐⭐⭐ |
| AI Token专区 | Token套餐展示、用量统计、续费提醒 | ⭐⭐⭐⭐⭐ |
| 帮助中心 | 使用教程、FAQ、客服联系 | ⭐⭐⭐ |
